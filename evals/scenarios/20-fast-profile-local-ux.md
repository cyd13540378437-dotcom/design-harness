# 20. Fast Profile Local UX

## Given

The project has `docs/design/project-memory/VISUAL_DESIGN.md`, `docs/design/VISUAL_DESIGN.md`, and a clear existing visual baseline.

## When

The user asks for a quick local UI improvement and explicitly says not to redesign the overall visual language.

## Then

- The Work Item uses `mode: "lightweight"` and `execution_profile: "fast"`.
- The state records Design Contract and Review Lens.
- The workflow may skip inapplicable visual-direction and prototype Gates.
- The workflow still records evidence and must use `completion-approval` before sealing.
