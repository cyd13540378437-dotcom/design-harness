# Dense Operator Console

## 适合

监控、运维、运营控制台、排班、交易、告警和数据密集工具。

## 可借鉴视觉特征

- 高密度卡片和表格。
- 明确状态色和优先级。
- 紧凑工具栏与筛选器。
- 信息面板之间的强对齐。

## 不应复制

- 不要为了密度牺牲可读性。
- 不要让所有信息同时争夺强调色。
- 不要使用真实监控产品图形或品牌元素。

## 配色倾向

深浅中性底色，状态色用于告警、成功、警告和信息。

## 布局倾向

多列仪表盘、固定工具栏、数据表和右侧详情检查面板。

## 组件倾向

表格、状态徽标、图表占位、筛选器、批量操作和告警列表。

## 不适合

消费级轻松体验、创作者工具或需要低认知负担的首次使用流程。
