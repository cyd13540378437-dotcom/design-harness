# Editorial Content System

## 适合

内容产品、知识库、课程平台、媒体后台、文档或文章管理系统。

## 可借鉴视觉特征

- 强排版层级。
- 栏目式网格和阅读节奏。
- 内容预览与编辑状态并重。
- 使用色块区分主题或栏目。

## 不应复制

- 不要把应用做成静态杂志封面。
- 不要牺牲后台操作效率。
- 不要复制任何真实媒体品牌版式。

## 配色倾向

高对比文字、克制背景、少量栏目强调色。

## 布局倾向

标题区、栏目网格、内容流、编辑或审核侧栏。

## 组件倾向

文章卡、栏目标签、状态筛选、编辑队列、阅读预览。

## 不适合

纯监控控制台、强交易工具或非常游戏化的消费应用。
