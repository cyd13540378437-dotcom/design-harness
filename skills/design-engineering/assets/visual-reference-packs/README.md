# Visual Reference Packs

These packs are self-made visual samples for Codex Design Harness. They help users choose concrete visual evidence when they do not have their own reference images.

Rules:

- Use these as inspiration, not as final UI assets.
- Do not treat a pack as a brand system.
- Do not copy third-party product screenshots, logos, or proprietary layouts into this directory.
- When a pack is used, record the selected pack and observable analysis in `STATE.md`.
- Approved results should be summarized in `docs/design/VISUAL_DESIGN.md`.

Available packs:

- `clean-minimal-product`
- `dense-operator-console`
- `warm-creator-tool`
- `editorial-content-system`
- `playful-consumer-app`
