# 糖果粉橙活力

## 用户可见摘要

适合高情绪参与和传播，但要控制视觉噪声。

## 色彩气质

这套色卡的主要气质是：年轻、活跃、社交、乐观。它在视觉上适合建立清晰的第一印象，同时保留足够的产品可用性边界。用于正式界面时，应把颜色当作产品语义和状态层级的一部分，而不是单纯装饰。

## 适合的产品语义

- 社交
- 活动
- 年轻消费
- 内容互动
- 轻游戏

## 不适合的产品语义

- 企业后台
- 医疗
- 金融安全场景
- 不适合作为长期阅读、医疗/心理健康、低压力办公产品的大面积底色。

## 界面落地建议

适合用于 onboarding、创作入口、社区互动和活动氛围；主流程里要控制彩色面积，避免降低信息判断效率。

建议把最深色阶用于文字、导航、主行动或视觉锚点，把中间色阶用于链接、图表和模块重点，把最浅色阶用于页面底色、卡片底色和空状态。彩色面积越大，越需要降低饱和度或提高留白比例。

## Token 转化建议

深粉做品牌锚点，玫粉/甜橙做活动行动，浅粉白只做轻背景。

可优先映射为：

```yaml
color.primary: primary
color.secondary: secondary
color.accent: accent
color.surface.tint: soft_background
color.background: page_background
```

## 无障碍与可读性风险

风险等级：高。

- 高风险：强饱和或强对比颜色容易造成疲劳，应限制在行动、警示或短时高亮。
- 暗色底、霓虹色、红色或粉橙色上的小字号文本必须做对比度和眩光检查。

## visual-direction-approval 展示话术

这是一套年轻、活跃、社交、乐观的配色方向。适合高情绪参与和传播，但要控制视觉噪声。适合：社交、活动、年轻消费、内容互动、轻游戏。不适合：企业后台、医疗、金融安全场景；实现时需要按风险等级检查文字、按钮、边界和状态反馈的对比度。
