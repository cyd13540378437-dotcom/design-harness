# 象牙黑金精品

## 用户可见摘要

适合强调品质和稀缺感，但需要控制可读性。

## 色彩气质

这套色卡的主要气质是：高级、克制、精品、仪式感。它在视觉上适合建立清晰的第一印象，同时保留足够的产品可用性边界。用于正式界面时，应把颜色当作产品语义和状态层级的一部分，而不是单纯装饰。

## 适合的产品语义

- 高端消费
- 会员
- 内容杂志
- 精品电商

## 不适合的产品语义

- 儿童
- 医疗
- 需要平价亲和的服务
- 不适合不做层级控制就直接铺满复杂后台；需要在品牌感和可读性之间做取舍。

## 界面落地建议

适合大留白、强排版、少量高质感强调色；不要把金色、红棕或深色铺满复杂操作界面。

建议把最深色阶用于文字、导航、主行动或视觉锚点，把中间色阶用于链接、图表和模块重点，把最浅色阶用于页面底色、卡片底色和空状态。彩色面积越大，越需要降低饱和度或提高留白比例。

## Token 转化建议

黑色做标题与高端锚点，金色只做少量强调，象牙白承载大面积阅读。

可优先映射为：

```yaml
color.primary: primary
color.secondary: secondary
color.accent: accent
color.surface.tint: soft_background
color.background: page_background
```

## 无障碍与可读性风险

风险等级：中。

- 中等风险：浅色底上的彩色小字、细线和禁用态需要单独验证对比度。
- 主 CTA 建议优先使用最深色阶或高饱和强调色，避免只用浅色承担关键行动。

## visual-direction-approval 展示话术

这是一套高级、克制、精品、仪式感的配色方向。适合强调品质和稀缺感，但需要控制可读性。适合：高端消费、会员、内容杂志、精品电商。不适合：儿童、医疗、需要平价亲和的服务；实现时需要按风险等级检查文字、按钮、边界和状态反馈的对比度。
