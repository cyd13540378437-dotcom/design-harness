# 猩红信号行动

## 用户可见摘要

适合需要强行动刺激的场景，慎用于主产品长期底色。

## 色彩气质

这套色卡的主要气质是：紧迫、行动、冲击、警示。它在视觉上适合建立清晰的第一印象，同时保留足够的产品可用性边界。用于正式界面时，应把颜色当作产品语义和状态层级的一部分，而不是单纯装饰。

## 适合的产品语义

- 活动发布
- 运动
- 竞赛
- 紧急状态
- 强 CTA

## 不适合的产品语义

- 心理健康
- 长时间办公
- 低压力工具
- 不适合作为长期阅读、医疗/心理健康、低压力办公产品的大面积底色。

## 界面落地建议

适合强入口、发布页、夜间模式和沉浸式产品；慎用于全站长期办公体验。

建议把最深色阶用于文字、导航、主行动或视觉锚点，把中间色阶用于链接、图表和模块重点，把最浅色阶用于页面底色、卡片底色和空状态。彩色面积越大，越需要降低饱和度或提高留白比例。

## Token 转化建议

猩红只做行动/警示，深红黑做标题，浅红底用于短时提示，不做长期大底色。

可优先映射为：

```yaml
color.primary: primary
color.secondary: secondary
color.accent: accent
color.surface.tint: soft_background
color.background: page_background
```

## 无障碍与可读性风险

风险等级：高。

- 高风险：强饱和或强对比颜色容易造成疲劳，应限制在行动、警示或短时高亮。
- 暗色底、霓虹色、红色或粉橙色上的小字号文本必须做对比度和眩光检查。

## visual-direction-approval 展示话术

这是一套紧迫、行动、冲击、警示的配色方向。适合需要强行动刺激的场景，慎用于主产品长期底色。适合：活动发布、运动、竞赛、紧急状态、强 CTA。不适合：心理健康、长时间办公、低压力工具；实现时需要按风险等级检查文字、按钮、边界和状态反馈的对比度。
