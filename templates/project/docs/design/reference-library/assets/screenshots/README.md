# Screenshots

此目录用于存放项目团队自行收集或截取的视觉参考图片。

建议按终端存放：

- `web-app/`：桌面 Web 应用、SaaS Dashboard、Web 编辑器。
- `mobile-app/`：iOS / Android 原生 App。
- `responsive-web/`：移动 Web、响应式官网或 Web App。
- `desktop-app/`：macOS / Windows 桌面应用。
- `tablet/`：平板端界面。

每组截图建议配套一个同名 `.md` 说明文件，记录：产品、来源 URL、截取日期、终端、页面类型、可借鉴点、不借鉴点和版权边界。
