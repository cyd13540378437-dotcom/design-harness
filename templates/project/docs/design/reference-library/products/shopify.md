# Shopify

## 适合参考的任务

商家后台、运营工具、订单系统

## 可借鉴

移动端后台需要把关键指标、待办和最近订单放在一屏内；编辑流程要非常稳。

## 不应借鉴

不要复用第三方品牌资产、图标、插画、文案或完整页面布局。

## 终端判断

- Primary surface: `web-app`
- Candidate surfaces: `web-app`, `mobile-app`
