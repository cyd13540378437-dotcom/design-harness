# Wise

## 适合参考的任务

涉及价格、费率、额度、时效解释的产品

## 可借鉴

把费用、汇率、到账时间显性化；用步骤条降低跨境金融焦虑。

## 不应借鉴

不要复用第三方品牌资产、图标、插画、文案或完整页面布局。 若当前任务是 Web/桌面端，不得直接照搬移动端导航、底部栏、手势或屏幕比例；只能抽象借鉴信息层级、反馈机制和文案语气。

## 终端判断

- Primary surface: `mobile-app`
- Candidate surfaces: `mobile-app`, `responsive-web`, `web-app`
