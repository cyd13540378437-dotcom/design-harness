# Medium

## 适合参考的任务

内容阅读、知识社区、博客产品

## 可借鉴

字体、行宽、标题层级比装饰更重要；会员和推荐应不破坏阅读节奏。

## 不应借鉴

不要复用第三方品牌资产、图标、插画、文案或完整页面布局。

## 终端判断

- Primary surface: `responsive-web`
- Candidate surfaces: `responsive-web`, `mobile-app`, `web-app`
- 选择它作为参考时，必须先确认当前 Work Item 的目标终端与页面类型是否匹配。
