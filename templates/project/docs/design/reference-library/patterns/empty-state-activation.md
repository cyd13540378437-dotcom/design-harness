# 空状态激活

## 何时适用

新账号、无数据、无权限或筛选无结果时，界面不能只显示空白。

## 可借鉴

- 解释为什么为空
- 给出一个最小下一步
- 必要时提供模板/示例数据

## 不应借鉴

- 不要用大段营销文案代替行动
- 不要让空状态变成死路

## QA 检查点

- 无数据、无权限、无结果三类空状态区分
- CTA 可用
- 移动端文案不拥挤
