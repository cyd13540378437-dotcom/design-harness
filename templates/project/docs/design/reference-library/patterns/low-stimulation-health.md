# 低刺激健康视觉

## 何时适用

用户处于压力、健康、睡眠、心理或陪伴型场景。

## 可借鉴

- 低饱和配色、宽松间距、柔和插画
- 文案像陪伴而不是命令
- 播放/训练页减少不必要信息

## 不应借鉴

- 不要使用过强对比和焦虑型数字表达
- 不要把健康反馈做成竞争惩罚

## QA 检查点

- 深浅主题可读性
- 危险/异常信息仍足够醒目
- 空状态不制造挫败感
