# 专业工具高密度工作台

## 何时适用

熟练用户需要快速扫描任务、状态、消息、对象关系和下一步。

## 可借鉴

- 侧边导航、列表/表格、状态标签、快捷入口
- 高密度但用分组和层级保持可扫读
- 主操作与状态变化靠近对象

## 不应借鉴

- 不要让新用户一进来就面对无解释的高密度界面
- 移动端只能借鉴对象层级，不能照搬布局

## QA 检查点

- 1440 宽度首屏可完成主要扫描
- 390 宽度有合理降级
- 长名称、空列表、权限不足可处理
