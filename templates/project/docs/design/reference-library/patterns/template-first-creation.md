# 模板先行降低门槛

## 何时适用

用户面对空白画布、复杂配置或创作任务时容易不知道从何开始。

## 可借鉴

- 模板缩略图让用户先看到结果
- 按目标/行业/复杂度分类
- 允许从模板开始后再自由编辑

## 不应借鉴

- 不要用模板替代真正的产品结构
- 不要让模板选择成为必须步骤

## QA 检查点

- 无模板/搜索无结果时有替代创建路径
- 模板预览和最终结果一致
- 模板分类不会过度营销化
