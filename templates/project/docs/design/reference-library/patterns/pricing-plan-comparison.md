# 定价与套餐比较

## 何时适用

用户需要选择套餐、理解限制、升级或取消。

## 可借鉴

- 按用户目标组织套餐差异
- 价格、限制、权益和 CTA 同屏
- 解释 trial、席位、用量和取消规则

## 不应借鉴

- 不要隐藏关键限制
- 不要用对比表制造误导性默认选择

## QA 检查点

- 移动端套餐表可比较
- 价格地区/周期变化可解释
- 升级/降级影响明确
