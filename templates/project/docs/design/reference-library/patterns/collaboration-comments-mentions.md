# 协作评论与 @ 提及

## 何时适用

多人需要围绕同一个对象、页面、视频、文件或任务讨论并形成行动。

## 可借鉴

- 评论锚定到具体对象
- @ 提及和通知形成闭环
- 已解决/未解决状态降低噪音

## 不应借鉴

- 不要把评论流和系统日志混在一起
- 不要过度打扰非相关成员

## QA 检查点

- 通知可追踪
- 权限变化时评论可见性正确
- 长线程可折叠
