# 深色内容沉浸

## 何时适用

内容资产本身具有强视觉/音视频吸引力，界面应让位给内容。

## 可借鉴

- 深色背景突出封面/视频/海报
- 播放控制和导航 chrome 尽量低噪音
- 推荐内容形成节奏

## 不应借鉴

- 不要让暗色导致长文本可读性下降
- 不要在工具型场景盲目追求沉浸

## QA 检查点

- 低亮度环境可读
- 焦点状态清晰
- 海报缺失时有占位方案
