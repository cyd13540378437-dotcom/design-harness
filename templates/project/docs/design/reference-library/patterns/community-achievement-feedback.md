# 社区成就反馈

## 何时适用

用户愿意分享“我做到了”或“我发现了”，社区反馈能强化动机。

## 可借鉴

- 成就卡、可分享截图、评论/点赞围绕核心内容
- 社区反馈与个人进度结合
- 默认保护隐私

## 不应借鉴

- 不要把私人健康/财务数据变成默认公开
- 不要让竞争压过自我成长

## QA 检查点

- 分享前可预览
- 敏感信息默认隐藏
- 无社交反馈时也有个人价值
