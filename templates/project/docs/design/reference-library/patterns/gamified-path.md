# 游戏化路径

## 何时适用

产品需要持续召回、进度感、打卡或训练动力。

## 可借鉴

- 路径、连胜、徽章、即时反馈
- 正反馈语言明确但不过度夸张
- 下一步任务永远清楚

## 不应借鉴

- 不要把奖励做得廉价或喧宾夺主
- 不适合所有 B 端效率场景

## QA 检查点

- 失败/中断时不羞辱用户
- 奖励不会挡住主任务
- 新用户能理解路径含义
