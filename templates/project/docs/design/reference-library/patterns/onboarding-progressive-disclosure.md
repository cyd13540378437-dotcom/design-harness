# 渐进式新手引导

## 何时适用

新用户需要理解价值、完成首个结果，而不是先学习完整系统。

## 可借鉴

- 先让用户得到小结果，再逐步开放复杂能力
- 用示例数据和模板降低空白感
- 引导可以跳过和恢复

## 不应借鉴

- 不要把引导做成强制教程
- 不要提前暴露所有设置

## QA 检查点

- 跳过后仍能找到下一步
- 回访用户不被反复打扰
- 无数据状态可激活
