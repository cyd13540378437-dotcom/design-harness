## 完整性摘要

- 商业分析保持独立，不包含跨模块 Trigger、Downstream View 或共享 Work Item。
- Decision Case 仅在 `decision-cases/` 保留一份权威文件。
- 用户可见结论统一使用中文 `DECISION_SUMMARY.md`。
- 多模型结果统一进入中文 `DECISION_SYNTHESIS.md` 后再生成摘要。
- 所有项目人读 Markdown 输出均有简体中文内容。
- 过程 artifacts 只进入 Work Item `artifacts/`，没有 `intake/` 目录。
