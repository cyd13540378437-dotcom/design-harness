# Source of Truth / 权威源

| 知识类型 | 唯一权威源 |
|---|---|
| Decision Case | `decision-cases/BA-DC-xxx.md` |
| BA Work Item 状态 | `work-items/BA-xxx/STATE.md` |
| 用户 D2/D3 决定 | `work-items/BA-xxx/DECISION_NOTES.md` |
| 用户可见阶段结论 | `work-items/BA-xxx/DECISION_SUMMARY.md` |
| 决策关系 | `decision-network/DECISION_NETWORK.yml` |
| 证据索引 | `evidence/EVIDENCE_LEDGER.yml` |
| 决策复审 | `reviews/BA-DR-xxx.md` |

Work Item 不得复制第二份 Decision Case。Process Package 是过程压缩，不是长期 Case 的替代。

所有 Markdown 权威输出必须有简体中文内容。
