# 最终结论契约（Final Conclusion Contract）v0.1.4

## 1. Choice Request 模式

说明当前选择、选项、推荐、后果、Assurance、改变条件，并邀请接受 / 拒绝 / 修改 / 暂缓。

## 2. Decision Confirmed 模式

说明用户已经确认什么、接受了什么代价、哪些声明仍受限制、何时复审。不得再次要求同一选择。

## 3. Post-decision Clarification 模式

先确认决定已记录，再对术语、阶段目的或证据边界做概念级解释；明确具体实施细节不属于当前 Case；最后进入 completion-review。

## 4. Review 模式

说明为什么重开、哪一项后果或假设改变，以及新的 Choice Set / 推荐。
