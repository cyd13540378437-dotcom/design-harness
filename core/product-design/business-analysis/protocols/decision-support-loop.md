# Business Analysis Decision Support Loop v0.1.1

```text
1. 完整保存第一次输入
2. 生成 Input Understanding Snapshot
3. 用户确认理解或系统记录 provisional 假设
4. 识别入口模式并生成 Analysis Routing Record
5. 创建 / 继续 / 复审 Decision Case，并绑定 BA Work Item
6. 通过 Next Best Question 收集最低必要信息
7. 接收、分类和筛选证据
8. 选择 Analysis Model Cards
9. 执行 Analysis Runs 与 Cross-model Synthesis
10. 执行反方分析
11. 执行 Decision Assurance Check
12. 生成选择题式结论或下一步选择
13. 记录用户决定
14. 更新 Decision Case、Decision Network、Evidence Ledger 或 Decision Review
15. 输出 Analysis Output 和 Decision Process Package
```

每次只推进到下一个合理决策状态；不以“填满模板”为完成标准。
